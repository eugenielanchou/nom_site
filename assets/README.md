# Images attendues ici

Le site (`index.html`, bilingue FR/EN) référence ces fichiers, à déposer dans ce dossier
`assets/` avec exactement ces noms. Tant qu'un fichier n'est pas présent, la page affiche
automatiquement un cadre "Photo à venir" à la place (pas d'image cassée).

## Logos institutionnels (footer)
- `logo-crnl.png`
- `logo-inserm.png`
- `logo-epfl.png`
- `logo-wisconsin.png`
- `logo-uva.png`

Fond transparent de préférence, hauteur ~150-200px.

## Photos du lieu (section "Le lieu" / "The Venue")
- `venue-landscape.jpg` — vue extérieure / paysage (ex. la photo aérienne du corps de ferme)
- `venue-room.jpg` — exemple de chambre
- `venue-dining.jpg` — salle commune / restaurant

Format paysage, ~1200×800px suffisant (elles sont recadrées en 4:3 sur le site).
